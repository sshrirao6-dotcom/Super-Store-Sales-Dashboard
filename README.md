# Super-Store-Sales-Dashboard

1. Project Title:
Sales Analytics Dashboard
2. Short Description / Purpose:
This project aims to analyze sales performance data and provide interactive visual insights. The
purpose is to help businesses track revenue trends, product performance, and customer behaviour
through a centralized dashboard.
3. Tech Stack:
- Python
- Excel
- Oracle SQL
- Power BI
4. Data Store:
Oracle SQL Database and Excel files used for initial raw data storage.
5. Features / Highlights:
- Interactive Power BI dashboard
- KPI tracking (Sales, Profit, Growth%)
- Product-wise, region-wise, and customer-wise analytics
- Automated data cleaning scripts in Python
6. Example Problem:
“Identify why sales dropped by 15% in Q3 and which product category contributed the most to the
decline.”
